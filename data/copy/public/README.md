# RAT-CKVN public settings

Các gói trong thư mục này chỉ chứa rule, risk, watchlist, indicator và preset chiến thuật.
Chúng không chứa `.env`, API key, token, số tài khoản, Telegram, lịch sử lệnh hay context riêng tư.

Thư mục `../private/` không được đưa lên GitHub và phải tự sao chép sang máy mới.
